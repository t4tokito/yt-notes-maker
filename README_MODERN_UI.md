## 📊 Usage Examples

### Basic Component Usage
```javascript
import { ModernUI } from './components/ModernUI';

<ModernUI.ModernCard>
  <ModernUI.ModernStatCard
    title="Notes"
    value="24"
    icon="📝"
    color={COLORS.primary}
  />
</ModernUI.ModernCard>
```

### Navigation Usage
```javascript
<ModernUI.ModernBottomNav
  activeTab={activeTab}
  onTabPress={setActiveTab}
/>
```
